Text: Time and Mark

Time

The time tag can be used to make dates and times. Such as Tuesday 1st June 2021 In addition to dates you can also put times using a 24 hour clock. For example: 12:45

Mark

You can highlight text using the mark tag.

Conditional Comments

Conditional Comments are just HTML comments that are used by older versions of Internet Explorer that are meant to help it produce the output desired for the web page.
They are also used in CSS to fix holes in these older browsers display capabilities.
Before using them think about what the users are going to be using and what makes the most sense for you to support. For example, the vast majority of people aren't using Internet Explorer 6 anymore.

Tables: Columns, Headers, and Footers

Columns, Caption, Header, Footer

The colgroup tag is used to help style the table with the use of CSS. You can give each column a class and use that class in CSS to style just that column.
You can use the span tag if you want multiple columns to look the same. (Don't use a col tag for the columns that the span is including.)
The caption tag is always used at the beginning of the table. CSS caption-side: left, right, or bottom will move it where you would expect.
The thead, tfoot, tbody tags allow you to separate the table into header, footer, and body parts.

Input Types

Search, Telephone, URL, Email, Numbers, Ranges, Dates, Times, and Color

  • Search is mostly used as a CSS style but also for a search query text box.
  • Can also use these attributes to do the same thing:
    • tel - Phone Numbers
    • url - Web Addresses
    • email - Email Addresses
  • In CSS use the valid/invalid pseudo classes to style these fields based off if they are written correctly.
  • "Number" and "step" types can be used to cycle through numbers in a text box.
  • They can also have min and max values. CSS valid and invalid are able to be used.
  • You can also set the type to "datetime", "date", "month", "week", "time", and "datetime-local"
  • If you want to do color through HTML and not CSS set type to "color" with a 6 digit hex representation of the color.
