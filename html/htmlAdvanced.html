Text: Time and Mark

Time

The time tag can be used to make dates and times. Such as Tuesday 1st June 2021 In addition to dates you can also put times using a 24 hour clock. For example: 12:45

Mark

You can highlight text using the mark tag.

Conditional Comments

Conditional Comments are just HTML comments that are used by older versions of Internet Explorer that are meant to help it produce the output desired for the web page.
They are also used in CSS to fix holes in these older browsers display capabilities.
Before using them think about what the users are going to be using and what makes the most sense for you to support. For example, the vast majority of people aren't using Internet Explorer 6 anymore.
