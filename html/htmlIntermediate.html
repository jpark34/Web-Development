Span and Div

Span and div both apply no meaning to text but allow for useful changes when used with CSS.
Span is in-line and used for a small chunk (Stays on the same line).
Div is block-line and used to group larger chunks (Puts a br before and after).

Meta Tags

One of the meta tags can be used as many times as needed in the head element. They contain the attributes charset, name, http-equiv, and content. When name or http-equiv are used, the content attribute is used with them to apply the meta data itself.
