Shadows

You can apply shadows to both text and boxes to make them stand out more on the web page.

box-shadow is the property that will add a shadow.

You can also add an inner shadow by adding inset into the property.

There are 5 values in it (the last 3 are optional):

  1. horizontal offset - how far the shadow is nudged to the right (or left if it is negative)
  2. vertical offset - how far the shadow is nudged downwards (or upwards if it is negative)
  3. blue radius - the higher the value the less sharp the shadow
  4. spread distance - the higher the value the larger the shadow
  5. color - color of the shadow

Applying a text shadow is the same as with boxes. Just use the text-shadow property instead.

Text-shadow has the same properties as box-shadow does (horizontal offset, vertical offset, blue radius, and color). However, there is no spread distance or inset options.
